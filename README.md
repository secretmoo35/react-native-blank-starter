# react-native-blank-starter

React-native use native-base components

this project install

- native-base
- @expo/vector-icons
- react-navigation

UI components: https://nativebase.io/

##### Installation use react-native

- npm install -g create-react-native-app

Install https://expo.io/

##### Installation project

- npm install

##### Run project

- expo start

##### Build project

- expo build:status
- expo build:android
- expo build:ios
